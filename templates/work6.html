Mail sending script using Python

Sending a mail without browser

Using smtpib and email libaries, mails can be sent without any contact from browser. Smtplibis basically provides a client session object which is used to start the process by loging in and then sending the mail content. Email library is used for mentioning the sender and reciever details.

Instead of sending test messages in mails , we can also attach a html file in the mail using PAth module from pathlib library and template module from string is used to read the content of the html page. For sending multiple emails, we have to create loop with iterating thee recipient;s id
