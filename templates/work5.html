005 : Portfolio Website

Presented my resume in a descriting way with all the details.

All the details regarding professional career under one Website

Python's scope in creating a website is limited in providing a framwork for web development. Here i have used Flask MicroFramework as it is lightweight and can be operated easily in virtual environment. Template HTML and CSS files were downloaded from Unsplash. We edited the HTML pages and created as many pages required. A web server was creted using flask , which regulated the html pages based on the response

Home page, About, Work, and other pages were created for display purpose. Contact me Page was created as interactive page as we can recieve messaged from other people, and we cann store their email, subject and message in an CSV file. In server side, we used CSV module for python recieve and save the data, and added an response page after the user has submitted the message.
