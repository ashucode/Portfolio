004 : Web Scraper

A python sript for extracting data from a website

News flashes from a News site, without having to use browser

Surfing news website can be tedious , going through headlines and watching their score and selecting most authentic one for reading. For instance if we get some kind of script that automatically chooses the news headlines depening on their scores and extract the headlines and full news link would be helpful

BeautifulSoup is a great library that acts as a lightweight microframework, providing all the basic neccesities for scraping websites. the main reason it is not called a framework because we have to do many things from scratch which may be wee defined in framework. Using this library we extracted news, based on their scores, we saved their headlines and the links for full news. now if required, the links can be surfed
