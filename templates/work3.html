003 : Twitter Bot

This was a fun project, just to check out twitter api.

Basic Functionality Twitter Bot

Basic Twitter activities comprises surfing tweets, liking or retweeting them, or following people. Python was used for scripting this functinalities. From twitter developer platform, all the requird keys and token were extracted as they were required. Tweepy module was used in this.

As the keys and tokens were private, I saved them in a document and accessed them in the code to maintain privacy. We accessed the tokens and sent through the api for authentication. I can access all the tweets in the home page, check the trending tweets based on the key words that are provided and also limit the number of tweets we want to take, and follow the followers
